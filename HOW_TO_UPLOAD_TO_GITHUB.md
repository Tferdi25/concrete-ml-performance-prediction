# How to upload this project to GitHub

## Option 1: using Git commands

From your computer, open a terminal inside the project folder and run:

```bash
git init
git add .
git commit -m "Initial commit: machine learning for concrete performance prediction"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/concrete-ml-performance-prediction.git
git push -u origin main
```

Replace `YOUR-USERNAME` with your GitHub username.

## Option 2: using GitHub website

1. Create a new repository named `concrete-ml-performance-prediction`.
2. Upload all files from this folder.
3. Keep the repository public if you want to show it in your CV.
4. Add the GitHub link to your CV under Digital Skills or Research Outputs.

## Suggested CV line

> GitHub: Machine learning workflow for predicting concrete mechanical, durability, and environmental performance indicators using Python, Scikit-learn, Random Forest, Support Vector Regression, Gradient Boosting, and Gaussian Process Regression.

## Important note

The included dataset is synthetic. If you use experimental data later, mention the data source and update the README accordingly.
